# 🌐 Website Metadata Extractor

A simple Flask-based web application that scans a website's HTML source and extracts useful metadata.

---

## 🚀 Features

- Extract HTML Meta Tags  
- Get Page Title  
- Detect Generator Information  
- Find Author Name  
- Extract Hidden HTML Comments  
- List Linked Images  
- Find Downloadable PDF Files  
- Basic Technology Detection  
- Clean and Simple UI  
- Invalid URL Protection  

---

## 🛠️ Technologies Used

- Python 3  
- Flask  
- BeautifulSoup4  
- Requests  
- HTML  
- CSS
